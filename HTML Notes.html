Units

-Pixels (px)-

Pixels have an absolute size on any browser or screen

-Percent (%)-

Percent is relative to the parent element or screen size if there is no parent element.

-Ems (em)-

Ems are relative to the default text size size. If they are within a parent element, they will be relative to the parent's font size.

-Rems (rem)-

Rems are relative to the root text size. To control the root size in CSS, adjust the "font-size" property of the "HTML" selector.

-Viewport Units (vh, vw)

(vh) Viewport heights are a percentage of the screen as the height of the item.

(vw) Viewport widths are a percentage of the screen as the width of the item.
